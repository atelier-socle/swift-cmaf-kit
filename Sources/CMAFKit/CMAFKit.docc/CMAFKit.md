# ``CMAFKit``

Pure-Swift, enterprise-grade ISO Base Media File Format / CMAF / CENC implementation.

## Overview

CMAFKit is the container and codec foundation of the Atelier Socle streaming
ecosystem. It reads, writes, validates, and encrypts fMP4 / CMAF media.

This is a foundation library. It does not encode, decode, transport, fetch DRM
licences, or generate playlists.

> Important: CMAFKit 0.1.0 ships the complete feature set after the 12-session
> implementation plan. APIs documented here reflect the planned 0.1.0 surface.
